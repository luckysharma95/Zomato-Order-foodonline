# 🍽️ Zomato Clone - Food Ordering Website

A responsive **Zomato Clone** built using **HTML, CSS, and JavaScript**. This project replicates the basic UI and user experience of the Zomato website, including navigation, search functionality, restaurant cards, and modern responsive design.

---

## 🚀 Features

- ✅ Responsive Navigation Bar
- ✅ Attractive Hero Section
- ✅ Food Search Bar
- ✅ Restaurant Cards
- ✅ Add Restaurant Page
- ✅ Investor Relations Page
- ✅ Login & Signup UI
- ✅ Responsive Design for Desktop
- ✅ Pure HTML, CSS & JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/luckysharma95/Zomato-Order-foodonline.git
```

2. Open the project folder

```
cd Zomato-Order-foodonline
```

3. Open `index.html` in your browser.
